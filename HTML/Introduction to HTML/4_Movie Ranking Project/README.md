## Key Concepts
- Headings in HTML are defined using the `<h1>`, `<h2>`, and `<h3>` tags.
- Paragraphs in HTML are defined using the `<p>` tag.
- Void elements are self-closing tags like `<hr>` that donâ€™t require a closing tag.
